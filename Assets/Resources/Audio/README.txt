About the usage of Audio Assets.
[Game Background Music]: 8-bit Gameboy Songs Vol 1 (GB Studio)_Electromagnetic Waves
[GameOver SFX] select 14 - 作为关卡失败
[StageClear SFX] start3 - 音调渐升，可以作为通关

[ButtonPressed SFX] SFX_UI_Click_Designed_Metallic_Pop_Negative_Locked_1 稍微活泼一点，可以作为按钮选中
